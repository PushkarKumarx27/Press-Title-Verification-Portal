## 📌 Project Overview
The **Press Title Verification Portal** is an intelligent automated system designed to detect and validate title duplications. By utilizing **Natural Language Processing (NLP)** similarity techniques and fuzzy matching, this system automates manual screening processes and improves overall throughput for title verification.

## 🚀 Key Features
* **Automated Title Verification:** Uses NLP and fuzzy matching to detect duplicate or highly similar titles.
* **Efficiency Boost:** Automates up to 80% of the manual screening and verification process.
* **Secure Access:** Features a built-in login system with dedicated user and admin portals.
* **Large-Scale Data Handling:** Processes and validates large textual datasets (CSV and Excel formats) for accurate cross-comparison.

## 🛠️ Technology Stack
* **Backend:** Python (Flask)
* **Data Processing:** Pandas, NLP, Fuzzy Matching
* **Frontend:** HTML, CSS (Jinja Templates)
* **Database/Storage:** JSON (`users.json`, `pending_titles.json`), CSV, Excel

## 📁 Project Structure
* `app.py` - The main application executable file.
* `main.ipynb` - Jupyter Notebook containing the core NLP and fuzzy matching logic/testing.
* `templates/` - Contains the frontend HTML pages (`home.html`, `admin.html`, `login.html`, `index.html`).
* `Dataset.csv` / `.xlsx` - Training and testing data files.
